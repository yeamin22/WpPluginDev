== Description ==

WFK Menu Import/Export lets you export and import your menu using a user interface. you will get the link to import export admin page after the appearance/menu

== Installation ==

1. Download `wfk-menu-import-export.zip`
2. Unzip
3. Upload the `wfk-menu-import-export` folder to the `/wp-content/plugins/` directory
4. Activate the 
WFK Menu Import/Export
 Download
WFK Menu Import/Export plugin through the 'Plugins' menu in WordPress
5. Export/Import your menus via Appearance -> Menu Import/Export in the WordPress admin

== Frequently Asked Questions ==

Yes, there is also a menu importer with the plugin
